# Engineering Resource Management System (ERM)

## Objective
Manage engineers, their skills, availability, and project allocation using a role-based system.

## Roles
- Admin
- Manager
- Engineer

## MVP Features
### Authentication
- JWT-based login
- Role-based authorization

### Engineer Management
- Create, update, view engineers
- Assign skills

### Project Management
- Create and manage projects

### Allocation
- Allocate engineers to projects
- Prevent over-allocation (>100%)

### Dashboard
- Bench vs billable engineers
- Utilization percentage

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL/PostgreSQL
